# 0226.翻转二叉树

递归地将左右子树交换就行

先序遍历和后序遍历都行，中序遍历不行

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    void myswap(TreeNode* &x, TreeNode* &y){
        TreeNode *t = x;
        x = y;
        y = t;
    }
    TreeNode* invertTree(TreeNode* root) {
        if(root == NULL){
            return root;
        }
        myswap(root->left, root->right);

        invertTree(root->left);
        invertTree(root->right);
        return root;
    }
};
```

需要注意的是，交换的是左右的两个指针

直接用STL自带的swap

```cpp
swap(root->left, root->right)
```

引用方式交换两个指针的指向

```cpp
void myswap(TreeNode* &x, TreeNode* &y){
    TreeNode *t = x;
    x = y;
    y = t;
}
```

指针方式交换两个指针的指向

```cpp
void myswap(TreeNode* *x, TreeNode* *y){
    TreeNode *t = *x;
    *x = *y;
    *y = t;
}
```
